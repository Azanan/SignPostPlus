# SignPostPlus
PocketMine-MPでご利用できる看板関係のプラグインです

## 機能一覧
* 羽をもって看板を触るとUIが展開され、コピーや書き換えなどができるようになります。
* 木の棒で立て看板を触るとクルクルと回転します
